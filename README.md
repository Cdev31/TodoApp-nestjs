# Pasos para ejecutar el proyecto
1. Descargar dependencias
2. Configurar las variables de entorno

3. Ejecutar el contenedor de la base de datos
```
docker compose up -d
```
4. Ejecutar el proyecto
```
yarn start:dev
```
